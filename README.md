# Rock-Paper-Scissors

A simple Rock-Paper-Scissors game built with HTML, CSS, and JavaScript.

## Features
- Play vs CPU 
- Score tracking  
- Simple and responsive UI  

## How to Play
- Rock beats Scissors  
- Scissors beats Paper  
- Paper beats Rock  
- Same choice = Tie
